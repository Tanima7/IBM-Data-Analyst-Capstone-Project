# IBM Data Analyst Capstone Project

This repository contains my **IBM Data Analyst Capstone Project** along with an additional **web scraping analysis** to explore technology skill demand in job listings.

The project demonstrates an end-to-end data analysis workflow including **data cleaning, exploratory analysis, SQL querying, visualization, and web scraping**.

---

# Project Components

## 1. Developer Survey Analysis

This notebook analyzes developer survey data to understand technology trends, developer roles, and salary patterns.

### Key Steps
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- SQL-based analysis
- Data visualization

### Tools Used
- Python
- Pandas
- SQL
- Power BI
- Matplotlib

Notebook:  
`IBM_Data_Analyst_Capstone_Project.ipynb`

---

## 2. Job Market Web Scraping Analysis

To complement survey insights, job listing data was collected using **web scraping techniques**.

### Workflow
- Sent HTTP requests to retrieve webpage data
- Parsed HTML using BeautifulSoup
- Extracted job description text
- Performed keyword-based skill analysis
- Visualized technology demand trends

### Tools Used
- Python
- Requests
- BeautifulSoup
- Pandas
- Matplotlib

Notebook:  
`Job_Market_Web_Scraping_Analysis.ipynb`

---

# Key Insights

- Programming languages such as **Java and Python** appeared most frequently in the scraped job descriptions.
- Keyword analysis can help estimate **technology demand trends** from job postings.
- Combining survey analysis with job listing data provides a broader perspective on **developer skill demand**.
